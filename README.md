# Usage
<!-- usage -->
```sh-session
$ npm install -g wp-client
$ wp-client COMMAND
running command...
$ wp-client (-v|--version|version)
wp-client/0.0.0 linux-x64 node-v10.5.0
$ wp-client --help [COMMAND]
USAGE
  $ wp-client COMMAND
...
```
